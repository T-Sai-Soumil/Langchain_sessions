The standard .with_structured_output() function in LangChain relies on the model's native support for tool/function calling or JSON mode. Because many Hugging Face models (and the Hugging Face serverless Inference API itself) do not natively support these features or their specific payloads, attempting to use .with_structured_output() directly on ChatHuggingFace raises errors such as huggingface_hub.errors.BadRequestError
